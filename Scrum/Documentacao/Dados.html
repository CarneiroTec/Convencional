Visão Geral

O módulo Dados define a estrutura base genérica utilizada internamente por outros módulos da biblioteca Convencional, especialmente pelo módulo Arquivo. Fornece funções para inicializar e limpar instâncias desta estrutura com segurança.

Interface: Interface/Dados/Dados.int
Implementação: Fonte/Dados/Dados.nld
Testes: Teste/Unidade/Dados/TesteDados.nld
Exemplo: Exemplo/Dados/ExemploDados.nld

Estrutura Principal — Dados

Representa um bloco de dados com metadados de posição, tamanho e identificação de arquivo.

Campo Tipo Valor Inicial Descrição
ponteiro Caractere* NULO Ponteiro para o início dos dados no buffer
tamanho Inteiro 0 Tamanho atual dos dados válidos
base Caractere* NULO Ponteiro para a base do buffer alocado
tipo Inteiro 0 Tipo do fluxo (leitura, escrita, etc.)
arquivo Inteiro -1 Descritor do arquivo associado (−1 = nenhum)
reservaCaractere Inteiro -1 Caractere reservado para devolução (ungetc)
tamanhoReserva Inteiro 0 Tamanho do buffer de reserva
nomeArquivo Caractere* NULO Nome do arquivo associado à instância

Enumeração de Erro — CtecErroDados

Todos os retornos de erro do módulo são tipados via este enumerador.

Pseudônimo Enumeração { CTEC_ERRO_DADOS_SUCESSO = 0, // Operação concluída com sucesso CTEC_ERRO_DADOS_NULO = -1, // Ponteiro nulo fornecido CTEC_ERRO_DADOS_INVÁLIDO = -2 // Estado inválido detectado } CtecErroDados;

Interface do Gerenciador — InterfaceGerenciadorDados

Acesse sempre pela instância global GerenciadorDados.

inicialize

Inicializa todos os campos de uma instância Dados com seus valores padrão. Deve ser chamado antes de qualquer uso da estrutura.

GerenciadorDados.inicialize(Dados *dados);
  • dados — Ponteiro para a instância a ser inicializada. Se NULO, retorna sem efeito.
Retorno: Vazio
⚠️ Ponteiro nulo é aceito silenciosamente — sem falha.

limpe

Reinicia todos os campos de uma instância Dados para seus valores padrão. Indicado para uso após finalização do uso da estrutura.

GerenciadorDados.limpe(Dados *dados);
  • dados — Ponteiro para a instância a ser limpa. Se NULO, retorna sem efeito.
Retorno: Vazio
⚠️ Ponteiro nulo é aceito silenciosamente — sem falha.

Exemplo de Uso

#Inclua <Convencional.int> Inteiro Início() { Dados bloco; // Inicializar antes do uso GerenciadorDados.inicialize(&bloco); // ... usar bloco com GerenciadorArquivo, etc. ... bloco.tipo = 1; bloco.arquivo = 3; // Limpar após o uso GerenciadorDados.limpe(&bloco); Retorne 0; }
