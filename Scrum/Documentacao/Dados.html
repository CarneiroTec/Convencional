Visão Geral

O módulo Dados define a estrutura base genérica utilizada internamente por outros módulos da biblioteca Convencional, especialmente pelo módulo Arquivo. Fornece 21 funções organizadas em: construtora, destrutora, extensora, auxiliares e acessores (defina/obtenha) para todos os campos.

Interface: Interface/Dados/Dados.int
Implementação: Fonte/Dados/Dados.nld
Testes: Teste/Unidade/Dados/TesteDados.nld
Exemplo: Exemplo/Dados/ExemploDados.nld

Estrutura — Dados

Campo Tipo Valor Inicial Descrição
ponteiro Caractere* NULO Ponteiro para início dos dados
tamanho Inteiro 0 Tamanho dos dados válidos
base Caractere* NULO Base do buffer alocado
tipo Inteiro 0 Tipo do fluxo
arquivo Inteiro -1 Descritor de arquivo (−1 = nenhum)
reservaCaractere Inteiro -1 Caractere para devolução
tamanhoReserva Inteiro 0 Tamanho do buffer de reserva

Enumeração de Erro — CtecErroDados

Pseudônimo Enumeração { CTEC_ERRO_DADOS_SUCESSO = 0, CTEC_ERRO_DADOS_NULO = -1, CTEC_ERRO_DADOS_INVÁLIDO = -2 } CtecErroDados;

Gerenciador — dados

Instância global: dados (nome do tipo em minúsculo).

📌 Convenção: Dados → instância dados

crie

Construtor — retorna uma instância Dados com todos os campos inicializados.

Dados d = dados.crie();
Retorno: Dados — instância inicializada

destrua

Destrutor — zera todos os campos da instância. Aceita ponteiro nulo sem falha.

dados.destrua(&d);
  • dados — Ponteiro para instância a destruir
Retorno: Vazio

clone

Copia todos os campos de origem para destino.

dados.clone(&destino, &origem);
  • destino — Ponteiro da instância que receberá os valores
  • origem — Ponteiro da instância fonte

inicialize / limpe

Inicializa ou reinicia todos os campos para valores padrão. Ambas aceitam ponteiro nulo.

dados.inicialize(&d); dados.limpe(&d);

Acessores — defina_* / obtenha_*

Para cada campo da estrutura, existem funções defina_[campo] e obtenha_[campo]. Todos são null-safe: se o ponteiro for nulo, retornam o valor padrão do campo.

defina obtenha Tipo
dados.defina_ponteiro(&d, val) dados.obtenha_ponteiro(&d) Caractere*
dados.defina_tamanho(&d, val) dados.obtenha_tamanho(&d) Inteiro
dados.defina_base(&d, val) dados.obtenha_base(&d) Caractere*
dados.defina_tipo(&d, val) dados.obtenha_tipo(&d) Inteiro
dados.defina_arquivo(&d, val) dados.obtenha_arquivo(&d) Inteiro
dados.defina_reserva_caractere(&d, val) dados.obtenha_reserva_caractere(&d) Inteiro
dados.defina_tamanho_reserva(&d, val) dados.obtenha_tamanho_reserva(&d) Inteiro
dados.defina_nome_arquivo(&d, val) dados.obtenha_nome_arquivo(&d) Caractere*

Exemplo de Uso

#Inclua <Convencional.int> Inteiro Início() { // Criar via construtor Dados bloco = dados.crie(); // Definir campos via acessores dados.defina_tipo(&bloco, 1); dados.defina_arquivo(&bloco, 3); dados.defina_nome_arquivo(&bloco, "exemplo.txt"); // Estender para outra instância Dados cópia = dados.crie(); dados.clone(&cópia, &bloco); // Destruir ambas dados.destrua(&bloco); dados.destrua(&cópia); Retorne 0; }
