Visão Geral

O módulo Variáveis encapsula o suporte a argumentos variádicos (va_start, va_arg, va_copy, va_end) em uma interface NLD de alto nível.

Enumeração de Erro — CtecErroVariádicos

Pseudônimo Enumeração { CTEC_ERRO_VARIÁDICOS_SUCESSO = 0, CTEC_ERRO_VARIÁDICOS_FALHA = -1 } CtecErroVariádicos;

Estrutura — ListaVariádicos

Campo Tipo Descrição
deslocamentoInteiro Natural Inteiro Offset atual para inteiros
deslocamentoReal Natural Inteiro Offset para ponto-flutuante
deslocamentoEstouro Natural Inteiro Offset de estouro (união)
áreaEstouro Caractere* Área de estouro (união)
áreaRegistros Caractere* Base do quadro de chamada

Gerenciador — variádicos

📌 Convenção: Variáveis → instância Imutável variádicos
Função Assinatura Descrição
inicie variádicos.inicie(&lista, quadro) Inicia a lista variádica (va_start)
obtenha variádicos.obtenha(&lista, tipo, tam, alin) Obtém próximo argumento (va_arg)
copie variádicos.copie(&dest, &orig) Copia estado (va_copy)
finalize variádicos.finalize(&lista) Finaliza a lista (va_end)

Exemplo de Uso

Vazio soma(Inteiro n, ...) { ListaVariádicos lista; variádicos.inicie(&lista, (Vazio*)((Caractere*)&n + meça(Inteiro))); // Processar argumentos... variádicos.finalize(&lista); }
