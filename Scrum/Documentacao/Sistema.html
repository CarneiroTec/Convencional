Visão Geral

O módulo Sistema agrupa 19 funções fundamentais em 5 sub-áreas acessíveis por uma única instância global sistema com structs aninhadas.

Interface: Interface/Sistema/Sistema.int
Implementação: Fonte/Sistema/Sistema.nld
Testes: Teste/Unidade/Sistema/TesteSistema.nld
Exemplo: Exemplo/Sistema/ExemploSistema.nld

Enumeração de Erro — CtecErroSistema

Pseudônimo Enumeração { CTEC_ERRO_SISTEMA_SUCESSO = 0, CTEC_ERRO_SISTEMA_FALHA = -1 } CtecErroSistema;

sistema.terminal

Entrada e saída via terminal (stdin/stdout).

Função Assinatura Descrição
exiba_caractere sistema.terminal.exiba_caractere('A') Exibe um caractere, retorna o caractere escrito
exiba_formatado sistema.terminal.exiba_formatado("%d", x) Saída formatada (printf)
exiba_texto sistema.terminal.exiba_texto("Olá") Exibe uma string com quebra de linha
leia_caractere sistema.terminal.leia_caractere() Lê um caractere do teclado
leia_formatado sistema.terminal.leia_formatado("%d", &x) Leitura formatada (scanf)
leia_texto sistema.terminal.leia_texto(buf) Lê uma linha de texto

sistema.ambiente

Variáveis de ambiente do sistema operacional.

Função Assinatura Descrição
defina_variável sistema.ambiente.defina_variável("NOME", "valor") Define uma variável de ambiente
obtenha_variável sistema.ambiente.obtenha_variável("NOME") Retorna o valor (ou NULO se inexistente)

sistema.execução

Controle de fluxo do processo em execução.

Função Assinatura Descrição
aborte sistema.execução.aborte() Encerra o programa abruptamente
finalize sistema.execução.finalize(0) Encerra com código de saída
finalize_imediamente sistema.execução.finalize_imediamente(1) Encerra sem callbacks
suceda sistema.execução.suceda(func) Registra função para chamada na finalização

sistema.tempo

Funções para obtenção de tempo e pausas.

Função Assinatura Descrição
obtenha_tempo sistema.tempo.obtenha_tempo(&t) Segundos desde 1970 (epoch)
obtenha_relógio sistema.tempo.obtenha_relógio() Ticks do processador
aguarde sistema.tempo.aguarde(1000) Pausa por N milissegundos

sistema.memória

Alocação e liberação dinâmica de memória.

Função Assinatura Descrição
aloque sistema.memória.aloque(1024) Aloca bloco não-inicializado
aloque_zerado sistema.memória.aloque_zerado(10, 4) Aloca bloco zerado
realoque sistema.memória.realoque(ptr, 2048) Redimensiona bloco existente
libere sistema.memória.libere(ptr) Libera bloco alocado

Exemplo de Uso

#Inclua <Convencional.int> Inteiro Início() { sistema.terminal.exiba_texto("Olá, Mundo!"); sistema.terminal.exiba_formatado("2 + 3 = %d\n", 5); Inteiro *p = (Inteiro*)sistema.memória.aloque(meça(Inteiro) * 10); *p = 42; sistema.memória.libere(p); Longo agora = sistema.tempo.obtenha_tempo((Vazio*)0); sistema.terminal.exiba_formatado("Timestamp: %ld\n", agora); Retorne 0; }
