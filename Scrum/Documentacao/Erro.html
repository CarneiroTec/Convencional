Visão Geral

O módulo Erro encapsula o acesso ao sistema de erros da linguagem C (errno / strerror) em uma interface NLD padronizada. Define os códigos de erro padrão POSIX traduzidos para português e fornece um gerenciador para ler, definir e converter códigos de erro em mensagens textuais.

Interface: Interface/Erro/Erro.int
Implementação: Fonte/Erro/Erro.nld
Testes: Teste/Unidade/Erro/TesteErro.nld
Exemplo: Exemplo/Erro/ExemploErro.nld

Tipo Base — Erro

Pseudônimo para Inteiro. Todos os códigos de erro do sistema utilizam este tipo.

Pseudônimo Inteiro Erro;

Enumeração de Erro do Módulo — CtecErroErro

Erros internos do próprio módulo Erro.

Pseudônimo Enumeração { CTEC_ERRO_ERRO_SUCESSO = 0, // Operação concluída CTEC_ERRO_ERRO_FALHA = -1 // Falha interna } CtecErroErro;

Interface do Gerenciador — InterfaceGerenciadorErro

Acesse sempre pela instância global cujo nome é o nome do tipo em minúsculo: erro.

📌 Convenção: Erro → instância erro

obtenha_erro

Retorna o código de erro atual do sistema (errno).

Erro atual = erro.obtenha_erro();
Retorno: Erro — código de erro atual (0 = sem erro)

defina_erro

Define o código de erro atual do sistema.

erro.defina_erro(ERRO_PERMISSÃO_NEGADA);
  • valor — Código de erro a ser definido (use constantes ERRO_*)
Retorno: Vazio

mensagem

Converte um código de erro numérico em sua mensagem textual equivalente.

Caractere *msg = erro.mensagem(ERRO_ARQUIVO_MUITO_GRANDE);
  • erro — Código de erro inteiro
Retorno: Caractere* — mensagem descritiva do erro

Exemplo de Uso

#Inclua <Convencional.int> Inteiro Início() { // Definir um erro erro.defina_erro(ERRO_PERMISSÃO_NEGADA); // Obter o erro atual Erro atual = erro.obtenha_erro(); // Converter para mensagem legível Caractere *msg = erro.mensagem(atual); // Resetar o erro erro.defina_erro(0); Retorne 0; }
