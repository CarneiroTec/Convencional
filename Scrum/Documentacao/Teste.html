Visão Geral

O módulo Teste fornece uma framework leve de testes unitários com duas estruturas: Caso (caso individual) e Grupo (agrupamento de casos). Instâncias globais: caso e grupo.

Enumeração de Erro — CtecErroTeste

Pseudônimo Enumeração { CTEC_ERRO_TESTE_SUCESSO = 0, CTEC_ERRO_TESTE_FALHA = -1 } CtecErroTeste;

Estrutura — Caso

Campo Tipo Descrição
estado Lógico 1 = passou, 0 = falhou
arquivo Caractere* Nome do arquivo de teste
função Caractere* Nome da função de teste
linha Inteiro Linha relevante do teste
mensagem Caractere* Mensagem descritiva

Gerenciador — caso

Função Assinatura Descrição
crie caso.crie(arq, func, lin, msg) Cria caso com metadados explícitos
instancie caso.instancie() Cria caso com macros ARQUIVO, FUNÇÃO, LINHA
destrua caso.destrua(&c) Zera todos os campos

Estrutura — Grupo

Campo Tipo Descrição
nome Caractere* Nome do grupo de testes
listaCasos FunçãoTeste[256] Array de funções de teste
quantidadeCasos Inteiro Total de casos registrados
quantidadePassou Inteiro Quantidade que passou
quantidadeFalhou Inteiro Quantidade que falhou

Gerenciador — grupo

Função Assinatura Descrição
crie grupo.crie("Nome") Cria grupo com nome
adicione grupo.adicione(&g, func) Adiciona caso ao grupo
execute grupo.execute(&g) Executa e exibe resumo
destrua grupo.destrua(&g) Limpa o grupo

Exemplo de Uso

// Cada teste retorna um Caso Caso meu_teste() { Caso c = { .estado = 1, .arquivo = ARQUIVO, .função = FUNÇÃO, .linha = LINHA, .mensagem = "" }; Se (2 + 2 != 4) { c.estado = 0; c.mensagem = "Falhou: 2+2 != 4"; } Se (c.estado) { c.mensagem = "Passou: 2+2 == 4"; } Retorne c; } Inteiro Início() { Grupo g = grupo.crie("Meus Testes"); grupo.adicione(&g, meu_teste); grupo.execute(&g); grupo.destrua(&g); Retorne 0; }
