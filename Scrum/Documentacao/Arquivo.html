Visão Geral

O módulo Arquivo encapsula todas as operações de I/O em arquivo (fopen, fread, fprintf, etc.) em uma interface NLD. Arquivo é um pseudônimo de Dados.

Interface: Interface/Arquivo/Arquivo.int
Implementação: Fonte/Arquivo/Arquivo.nld
Testes: Teste/Unidade/Arquivo/TesteArquivo.nld (11 testes)
Exemplo: Exemplo/Arquivo/ExemploArquivo.nld

Enumeração de Erro — CtecErroArquivo

Pseudônimo Enumeração { CTEC_ERRO_ARQUIVO_SUCESSO = 0, CTEC_ERRO_ARQUIVO_FALHA = -1, CTEC_ERRO_ARQUIVO_NULO = -2 } CtecErroArquivo;

Gerenciador — arquivo

📌 Convenção: Arquivo → instância Imutável arquivo

Abrir / Fechar

Função Assinatura Descrição
crie arquivo.crie("nome.txt", "w") Cria/abre com modo
abra arquivo.abra("nome.txt", "r") Abre existente com modo
feche arquivo.feche(arq) Fecha arquivo aberto

Leitura / Escrita

Função Assinatura Descrição
leia arquivo.leia(buf, tam, qtd, arq) Leitura binária de blocos
escreva arquivo.escreva(buf, tam, qtd, arq) Escrita binária de blocos
despeje arquivo.despeje(arq) Força escrita do buffer (fflush)
escreva_formatado arquivo.escreva_formatado(arq, "%d", x) Escrita formatada (fprintf)
leia_formatado arquivo.leia_formatado(arq, "%d", &x) Leitura formatada (fscanf)

Posição

Função Assinatura Descrição
posicione arquivo.posicione(arq, 0, 0) Move cursor (fseek)
obtenha_posição arquivo.obtenha_posição(arq) Posição atual (ftell)
reinicie arquivo.reinicie(arq) Volta ao início (rewind)

Estado

Função Assinatura Descrição
fim_arquivo arquivo.fim_arquivo(arq) Verifica fim (feof)
erro_arquivo arquivo.erro_arquivo(arq) Verifica erro (ferror)
limpe_erro arquivo.limpe_erro(arq) Limpa indicadores (clearerr)

Caractere / Linha

Função Assinatura Descrição
obtenha_caractere arquivo.obtenha_caractere(arq) Lê um caractere (fgetc)
exiba_caractere arquivo.exiba_caractere('A', arq) Escreve um caractere (fputc)
obtenha_linha arquivo.obtenha_linha(buf, n, arq) Lê linha (fgets)
exiba_linha arquivo.exiba_linha("texto", arq) Escreve linha (fputs)
devolva_caractere arquivo.devolva_caractere('X', arq) Devolve ao buffer (ungetc)

Buffer / Temporário / Gerenciamento

Função Assinatura Descrição
defina_reserva arquivo.defina_reserva(arq, buf, modo, tam) Configura buffer (setvbuf)
configure_reserva arquivo.configure_reserva(arq, buf) Buffer padrão (setbuf)
temporário arquivo.temporário() Cria arquivo temporário anônimo
nome_temporário arquivo.nome_temporário(buf) Gera nome temporário
remova arquivo.remova("nome.txt") Remove arquivo
renomeie arquivo.renomeie("a.txt", "b.txt") Renomeia arquivo

Exemplo de Uso

#Inclua <Convencional.int> Inteiro Início() { Arquivo *arq = arquivo.crie("saida.txt", "w"); arquivo.escreva_formatado(arq, "Olá %s!\n", "Mundo"); arquivo.feche(arq); arq = arquivo.abra("saida.txt", "r"); Caractere buf[100]; arquivo.obtenha_linha(buf, 100, arq); arquivo.feche(arq); arquivo.remova("saida.txt"); Retorne 0; }
